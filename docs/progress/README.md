# Public Progress Reports

This directory contains evidence-based source reports for ArtSoul's public
build-in-public updates to Base and the wider community.

## Cadence

- Tuesday report: covers the preceding Saturday through Tuesday.
- Friday report: covers Wednesday through Friday.
- The first report in this cadence covers 2026-07-17 through 2026-07-21.
- The next report covers 2026-07-22 through 2026-07-24.

This produces two non-overlapping reports per week in the Europe/Warsaw time
zone. If a scheduled day has no material merged work, publish a short honest
status update instead of inventing progress.

## Evidence rules

- Use merged `main` changes and completed operational verification only.
- Do not describe a draft PR as shipped.
- Say explicitly when a feature is implemented behind a disabled flag.
- Do not include secrets, private wallet assignments, personal data, internal
  codenames, canon discussions or uncommitted funding.
- Keep protocol and economic claims consistent with the public repository.
- Link the repository or relevant public artifact when useful.

File names use `BASE_X_UPDATE_YYYY-MM-DD_TO_YYYY-MM-DD.md`.
