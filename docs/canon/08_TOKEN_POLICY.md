# Token Policy

ArtSoul is token-free for the current product.

## Not Allowed

- No token.
- No points.
- No airdrop logic.
- No staking logic.
- No token reward accounting.
- No passive-income promise.

## Allowed

- NFT artworks.
- Soulbound Genesis art NFTs.
- Soulbound Top-100k profile badge.
- Protocol and ecosystem treasuries as defined in the revenue canon.

Any future token requires an explicit canon amendment and is out of scope for this repository.
